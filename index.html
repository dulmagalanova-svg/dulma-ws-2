Блок 4 · 15 минут

Скиллы: два типа и установка

Project — это память ассистента. Скилл — конкретная инструкция для конкретной задачи. Грузятся в два разных места.

Тип 1
Кастомные скиллы — внутри Project
  • Текстовые .md файлы
  • Загружаются через Project → Add content
  • Работают только внутри этого проекта
  • tg-post-writer.md, instagram-content-creator.md, humanizer.md
Тип 2
Personal Skills — глобальные, с кодом
  • Выполняют код — рисуют PNG, создают файлы
  • Загружаются через Settings → Customize → Skills → «+»
  • Работают во всех чатах и проектах сразу
  • carousel-design.zip — рисует карусели в фирменном стиле
Установить carousel-design как Personal Skill:
  • Settings → Customize → Skills → «+»
  • Upload skill → выбери carousel-design.zip из раздачи
  • Найди в списке → включи тумблер справа
Делается один раз. После этого скилл работает во всех чатах автоматически.
+
Создать скилл с нуля — через Claude

Нужен скилл которого нет в раздаче — попроси Claude написать его:

Промпт для генерации скилла
Создай скилл (инструкцию в формате .md) для написания
email-рассылки для онлайн-школы.

Скилл должен включать:
— описание задачи
— структуру письма (тема, прехедер, тело, CTA)
— тон: дружелюбный, не продающий напрямую
— чего избегать: спам-слова, слишком длинные письма
— 2 примера готовых писем

Формат: markdown файл, готовый к загрузке в Claude Project.

Получаете файл → сохраняете как .md → загружаете в Project. Готово.

01
Что такое хороший лид-магнит
✅ Хороший лид-магнит:
  • Решает одну конкретную боль за 5–15 минут чтения
  • Короткий и прикладной — не учебник на 50 страниц
  • Даёт быстрый результат («вау, она реально шарит»)
  • Связан с темой платного продукта
❌ Плохой лид-магнит:
  • «Гайд по [огромная тема]» на 70 страниц
  • Общая теория без конкретных шагов
  • То что можно загуглить за 5 минут
Примеры по нишам:
  • Психолог: «5 техник самопомощи при панической атаке» — 4 страницы
  • Нутрициолог: «Готовое меню на неделю для занятых мам» — 6 страниц с рецептами
  • Маркетолог: «Чек-лист продающего лендинга — 17 пунктов» — одна страница
  • Финансист: «10 точек где вы теряете деньги в семейном бюджете» — 3 страницы

Разница
Чем отличается агент от воронки

Воронка — это автомат. Фиксированные сообщения по таймеру. Если человек спросит «сколько стоит?» — воронка не услышит, продолжит свои сообщения по графику. Человек уйдёт.

ИИ-агент — это менеджер. Слушает каждое сообщение. Понимает контекст. Отвечает по теме. Дожимает на продажу. Передаёт горячих лидов вам в Telegram.

Что умеет
Что умеет агент в Chatplace
  • Понимает текст, голосовые, кружки, изображения
  • Отвечает мгновенно, в течение 2–3 секунд
  • Помнит контекст переписки
  • Сохраняет имена, телефоны, email в карточку клиента
  • Передаёт уведомление в Telegram когда клиент готов покупать
  • Работает в Instagram, Telegram-бот, TikTok, Telegram Business
  • Отвечает на комментарии под постами и реакции на сторис (Instagram)
Мозг
Какой «мозг» у агента
  • Под капотом — Claude Sonnet 4.5 как основная модель
  • При перегрузке — автоматически переключается на ChatGPT-5
